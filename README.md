# Dating App 💖

A modern **Dating Application UI** built using **Kotlin and Jetpack Compose**.
This project demonstrates a clean and scalable Android UI architecture using modern Android development tools.

---

## 📱 Features

* Modern **Jetpack Compose UI**
* Clean and responsive layouts
* Swipe-style dating interface
* Notification UI
* Profile UI design
* Gradient buttons and custom components
* Reusable composables
* Material Design styling

---

## 🛠 Tech Stack

* **Kotlin**
* **Jetpack Compose**
* **Material Design**
* **Android Studio**

---

## 🚀 Getting Started

### 1 Clone the repository

```
git clone https://github.com/payelpaul121999/DatingApp.git
```

### 2 Open in Android Studio

* Open **Android Studio**
* Click **Open**
* Select the project folder

### 3 Run the application

Connect an Android device or emulator and click **Run ▶**
